# 甄嬛传辩论生成器

这是一个基于 Next.js 14 开发的现代化 Web 应用，用于生成甄嬛传风格的辩论内容。

## 技术栈

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Radix UI
- shadcn/ui
- Zustand
- React Query
- Jest

## 本地开发

1. 克隆项目
```bash
git clone [你的仓库地址]
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 打开浏览器访问 http://localhost:3000

## 部署

本项目使用 Vercel 进行部署，访问 [部署地址] 即可查看在线版本。

## 功能特性

- 生成甄嬛传风格的辩论内容
- 支持多种角色选择
- 响应式设计
- 暗色/亮色主题切换
- PWA 支持
